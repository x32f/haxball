## 使い方
この[テキストコード](URL)を全てコピーする<br>
メモ帳などを使い、[設定エリア]の!stopと!adminを任意の文字列に変更しておく<br>
ブラウザで[Haxball Headless Host](https://www.haxball.com/headless)を開く<br>
F12キーを押し、開発者ツールを開く<br>
コンソール欄にペーストし、Enter<br>
ロボット認証が出てくるのでチェック<br>
部屋のリンクが出る<br>
部屋のリンクを別タブで開けば部屋建て完了！

### 注意事項
Haxball Headless Hostのページをリロードしたり、閉じたりすると部屋が消えます。

### 主な機能
【自動チーム分け】試合終了後、自動でチーム分けを行います。!afk状態の人は除外します。<br>
【スタジアム変更】人数に応じて、自動的にスタジアムサイズを変更します。!afk状態の人は除外します。<br>
【離席キック】30秒離席で観戦者に、その後10分経過でキックします。<br>
【実況】 スーパーセーブやロングパスなどを感知して実況します。<br>
【テニスモード】 ボールが相手のゴールラインを超えたら得点になります。7点先取です。<br>
【ドッヂボール】 相手の蹴ったボールに当たると場外に飛ばされます。ボールが壁に当たるまで触れてはいけません。相手を全滅させると得点になります。2点先取です。

### 主なコマンド
【!stop】隠しコマンド。試合を中断します。[設定エリア]で任意の文字列に変更してください。<br>
【!admin】隠しコマンド。管理者権限を取得します。[設定エリア]で任意の文字列に変更してください。<br>
【!afk】観戦します。<br>
【!unafk】試合に戻ります。<br>
【!help】コマンド一覧を表示します。
